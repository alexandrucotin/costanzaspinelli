# Guida Media per il Sito

Questa guida spiega quali immagini e video inserire per completare il sito web.

## Immagini da Aggiungere

### 1. Hero Background (`/public/hero-bg.jpg`)

**Dimensioni consigliate**: 1920x1080px o superiore
**Tipo**: Immagine di palestra moderna, attrezzature fitness, o ambiente motivazionale
**Suggerimenti**:

- Usa siti come Unsplash, Pexels, o Pixabay per immagini gratuite
- Cerca: "gym equipment", "fitness training", "workout space"
- L'immagine verrà oscurata con overlay, quindi va bene anche con colori vivaci

**Link suggeriti**:

- https://unsplash.com/s/photos/gym-equipment
- https://www.pexels.com/search/fitness/

### 2. Foto Professionale Costanza (`/public/costanza-profile.jpg`)

**Dimensioni consigliate**: 800x800px (quadrata)
**Tipo**: Foto professionale in ambiente fitness
**Suggerimenti**:

- Abbigliamento sportivo professionale
- Sfondo neutro o palestra
- Espressione sorridente e professionale
- Buona illuminazione

### 3. Video Introduttivo (Opzionale)

**Formato**: MP4, max 50MB
**Durata**: 30-60 secondi
**Contenuto suggerito**:

- Presentazione breve di Costanza
- Mostra esercizi o ambiente di lavoro
- Messaggio motivazionale
- Call to action finale

## Come Sostituire le Immagini

1. Scarica le immagini che vuoi usare
2. Rinominale esattamente come indicato sopra
3. Metti i file nella cartella `/public`
4. Riavvia il server di sviluppo se necessario

## Note Importanti

- **Diritti d'autore**: Assicurati di avere i diritti per usare tutte le immagini
- **Ottimizzazione**: Comprimi le immagini prima di caricarle (usa TinyPNG.com)
- **Formato**: Preferisci JPG per foto, PNG per loghi/grafiche
- **Dimensioni**: Non superare 500KB per immagine per velocità di caricamento
